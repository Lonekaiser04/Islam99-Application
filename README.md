# 🌙 Islam99 App

Islam99 is a comprehensive Islamic mobile application designed to help Muslims stay connected with their faith in daily life. It integrates essential Islamic resources, tools, and features into a clean and user-friendly interface.

---

## 📱 Features

### 🕌 Core Features

* 📖 **Holy Quran** – Read Quran using JSON-based data
* 📜 **Hadith Collection** – Access Hadiths through integrated PDF resources
* 🤲 **Dua & Azkar** – Daily supplications and remembrance
* 🧿 **Aurad-e-Fatiha** – Special recitations

---

### 🕰️ Prayer & Time Features

* 🕋 **Prayer Times** – Based on real-time location APIs
* ⏳ **Countdown to Next Prayer**
* 📅 **Monthly Prayer Timetable**
* 🧭 **Qibla Direction** – Accurate compass integration

---

### 📆 Calendar & Date

* 🌙 **Hijri Calendar**
* 📅 **Gregorian Calendar**
* 🔄 **Dual Calendar Support**

---

### 🌟 Daily Inspiration

* 📖 **Verse of the Day**
* 📜 **Hadith of the Day**

---

### 📿 Islamic Tools

* 🧮 **Zakat Calculator**
* 📿 **Digital Tasbih Counter**
* 🕌 **99 Names of Allah (Asma-ul-Husna)**

---

### ⚙️ Additional Features

* ⚙️ **App Settings & Customization**
* 📍 Location-based services
* 🌐 Clean and responsive UI

---

## 🛠️ Technologies Used

* 💙 **Flutter** – Cross-platform UI framework
* 🎯 **Dart** – Programming language
* 📂 **JSON** – Quran data handling
* 📄 **PDF Integration** – Hadith collections
* 📍 **APIs** – Prayer time & location services
* 🧑‍💻 **Android Studio** – Development environment
